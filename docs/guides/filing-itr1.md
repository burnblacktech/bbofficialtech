# ITR-1 (Sahaj) — Step-by-Step Filing Guide

> For salaried individuals with income up to ₹50 lakh

## Who Should File ITR-1?
- Salaried individuals (one or more employers)
- Income from one house property
- Other income (interest, dividends up to ₹5,000)
- Agricultural income up to ₹5,000
- **Total income ≤ ₹50 lakh**

## Step-by-Step Flow on BurnBlack

### Step 1: Start Filing
1. Login → Dashboard → Click **"File ITR"**
2. System auto-creates ITR-1 filing for current AY (2026-27)
3. If PAN not verified → redirected to PAN verification first
4. If existing draft exists → resumes it

### Step 2: Personal Information
1. Click **"Personal Info"** in the left panel
2. Fields auto-filled from PAN verification (name, DOB — locked)
3. Fill remaining:
   - Gender, Email, Phone
   - Aadhaar (click "Verify via OTP" → enter OTP → auto-fills address)
   - Address (flat, city, state, pincode)
   - Residential Status (select "Resident" if lived in India 182+ days)
   - Employer Category (Private Sector for most)
   - Filing Status (Original for first filing of this AY)
4. Click **Save** → "✓ Personal info complete"

### Step 3: Salary Income
1. Click **"Salary"** in the left panel
2. Click **"Add Employer"**
3. Fill:
   - Employer name, TAN
   - Gross salary (from Form 16 Part B)
   - HRA received + rent paid + city (for HRA exemption calc)
   - LTA claimed
   - Professional tax deducted
   - TDS deducted (from Form 16 Part A)
4. **Shortcut**: Click "Upload Form 16" → auto-fills all salary data
5. Standard deduction (₹75,000) applied automatically

### Step 4: Other Income (Optional)
1. Click **"Other Income"** in the left panel (or add it via "+ Add income source")
2. Fill applicable fields:
   - Savings account interest
   - FD/RD interest
   - Dividend income
   - Family pension
   - Any other income
3. Agricultural income (if ≤ ₹5,000)

### Step 5: House Property (Optional)
1. Add "House Property" source if applicable
2. Select type:
   - **Self-occupied**: Enter home loan interest (max ₹2L deduction)
   - **Let-out**: Enter rent received, municipal taxes, loan interest
3. System computes net income/loss from house property

### Step 6: Deductions (Old Regime Only)
1. Click **"Deductions"** in the left panel
2. Fill applicable sections:
   - **80C** (₹1.5L limit): PPF, ELSS, LIC, EPF, tuition fees, home loan principal
   - **80CCD(1B)**: NPS contribution (₹50K additional)
   - **80D**: Health insurance (self + parents)
   - **80E**: Education loan interest
   - **80G**: Donations
   - **80TTA/80TTB**: Savings interest (₹10K/₹50K)
3. If New Regime selected → deductions section shows "Not applicable" message

### Step 7: Bank Details
1. Click **"Bank & Submit"** in the left panel
2. Fill:
   - Bank name
   - Account number
   - IFSC code
   - Account type (Savings/Current)
3. This is where refund will be credited

### Step 8: Review Tax Computation
1. The **Result card** (bottom of left panel) shows:
   - Total income
   - Tax liability (old vs new regime comparison)
   - TDS already paid
   - Refund or tax payable
2. Toggle between Old/New regime to compare
3. System recommends the regime with lower tax

### Step 9: Submit
1. Click **"Submit"** button
2. System validates:
   - All required fields filled
   - Income within ₹50L limit
   - Bank details valid
   - Email verified
3. On success → Filing submitted to ERI → Redirected to Submission Status

### Step 10: Post-Filing
1. **E-verify within 30 days** (Aadhaar OTP is fastest)
2. Download computation PDF for records
3. Track status on Submission Status page

## Tax Computation (AY 2026-27)

### New Regime (Default)
| Slab | Rate |
|------|------|
| 0 – ₹4L | 0% |
| ₹4L – ₹8L | 5% |
| ₹8L – ₹12L | 10% |
| ₹12L – ₹16L | 15% |
| ₹16L – ₹20L | 20% |
| ₹20L – ₹24L | 25% |
| ₹24L+ | 30% |

Rebate: If taxable income ≤ ₹12L → rebate up to ₹60,000 (effectively zero tax)
Standard deduction: ₹75,000

### Old Regime
| Slab | Rate |
|------|------|
| 0 – ₹2.5L | 0% |
| ₹2.5L – ₹5L | 5% |
| ₹5L – ₹10L | 20% |
| ₹10L+ | 30% |

Rebate: If taxable income ≤ ₹5L → rebate up to ₹12,500
Standard deduction: ₹75,000
All Chapter VI-A deductions available (80C, 80D, etc.)

### Additional
- Surcharge: 10% if income > ₹50L (unlikely for ITR-1)
- Health & Education Cess: 4% on (tax + surcharge)
